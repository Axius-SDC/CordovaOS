# CordovaOS
Cordova OS is one Django **project** containing ten Django **apps**, each generated by [SDCStudio](https://github.com/Axius-SDC/SDCStudio) from its own data model. The apps interoperate because they share the same SDC4 Reference Model (SDCRM) — no middleware, no API adapters, no ETL pipelines, no data mapping layer.
